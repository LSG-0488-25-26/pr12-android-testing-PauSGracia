# ANDROID TESTING
- Aquesta app va de probar el testing d'android utilitzant BOM i Espressoç

## UNIT TESTING
- Aquests testos serveixen per provar codi
- Primer es posa aquest tros de codi per fer que les variables del viewmodel es sincronitzin i crear un nou viewmodel cada vegada que es faci un test
<img width="354" height="179" alt="{A514B2E7-04F1-4767-A0FC-160170E3A283}" src="https://github.com/user-attachments/assets/14531321-90b5-4bc5-9c08-1cf50465891e" />

- Hi ha un test pe comprovar que les variables inicials estiguin correctes
- Hi ha un test per cada component, per veure que totes les funcionalitats funcionin bé

## INSTRUMENTAL TESTING
- Aquests testos serveixen per provar la UI i els components
- Hi ha un test pe comprovar que les variables inicials estiguin correctes
- Hi ha un test per cada component, per veure que els components de la pantalla funcionin bé
